QA for LIB2024

  1. Qus:What are the three components of Web?

    Ans:HTML, CSS, and Javascript

  2. Qus:Define version control and give one example.

    Ans:A system that records changes to a file over time so you can access each version later. Ex:Github

  3. Qus:What tag is used to create the title of a web page? Give and example

    Ans:<title> content in title </title>

    <title>Example here </title>

  4. Qus:What tag is used to create a paragraph in html? Give an example

    Ans:<p> content in paragraph </p>

    This is a paragraph

  5. Qus:Identify the odd one from these tags: <span>,<p>,<a>,<i> and state the reason

    Ans:<a> because it links the page to another site, where the other tags are simply formatting the text

  6. Qus:What are the tags required to create a navigation list in HTML? Give one example with two navigation links (https://algonquincollege.com, https://google.com)

    Ans:

    <nav>

    <ul>

    <li><a href="https://algonquincollege.com">Algonquin College </a>

    <li><a href="https://google.com">

    /nav

    • Algonquin College
    • Google
